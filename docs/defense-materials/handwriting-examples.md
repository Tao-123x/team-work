# Handwriting Examples

This file gives realistic handwritten-note examples for each member across the five agreed meetings.

Use them as a tone guide.

Do **not** copy every sentence word-for-word across all pages.

Best practice:

- keep each page to `5 to 8` short lines
- mix short English phrases with a few Chinese notes if natural
- use arrows, underlines, circles, and abbreviations
- make each member's focus look different

## Quick Variation Tips

To keep pages realistic, vary:

- sentence length
- English vs short Chinese notes
- whether a line starts with `need`, `check`, `agree`, `risk`, `next`
- whether you use arrows like `->`

Good small variations:

- `need clearer MVP`
- `risk: route confusion`
- `agree core flow first`
- `next -> fix main status path`
- `老师反馈后必须收口`
- `demo only show happy path`

## Best Meetings To Write First

If the team is short on time, write in this order:

1. Meeting 1
2. Meeting 2
3. Meeting 5
4. Meeting 4
5. Meeting 3

That order matches what the teacher is most likely to care about first.
